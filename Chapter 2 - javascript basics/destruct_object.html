Destructuring assignment allows you to locally scope fields within an object and to declare which values will be used.

Code: 

	const sandwich = {
		bread: "dutch crunch",
		meat: "tuna",
		cheese: "swiss",
		toppings: ["lettuce", "tomato", "mustard"]
	};

	const { bread, meat } = sandwich;

	console.log(bread, meat);
Output: 

	dutch crunch tuna
