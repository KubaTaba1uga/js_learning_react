Destructuring assignment allows you to locally scope values within a data structures.

Code: 

	 let list = [1,2,3,4,5,6];
	 let [first, second] = list;
	 console.log(list);
	 console.log("First:",first);
	 console.log("Second:", second);
Output: 

	Array(6) [ 1, 2, 3, 4, 5, 6 ]

	First: 1 
	Second: 2
