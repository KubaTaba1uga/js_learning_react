The spread operator is three dots ( ... ) that perform several different tasks. First, the spread operator allows us to combine the contents of arrays.

Code: 

	const peaks = ["Tallac", "Ralston", "Rose"];

	const canyons = ["Ward", "Blackwood"];

	const tahoe = [...peaks, ...canyons];

	console.log(tahoe.join(", "));
Output: 

	Tallac, Ralston, Rose, Ward, Blackwood

To copy content of array.

Code: 

	const peaksCopy = [...peaks];
	console.log(peaks.join(", "))
Output: 

	Tallac, Ralston, Rose

The spread operator can also be used to get the remaining items in the array.

Code: 

	 const [first, ...others] = peaks;
	 console.log(others.join(", "))
Output: 

	Ralston, Rose

We can also use the three-dot syntax to collect function arguments as an array. When used in a function, these are called rest parameters.

Code: 


	 function print_args(...args) {
		args.forEach(console.log);
	 };

	 print_args("Tallac", "Ralston", "Rose");
Output: 


	Tallac 0 Array(3) [ "Tallac", "Ralston", "Rose" ]

	Ralston 1 Array(3) [ "Tallac", "Ralston", "Rose" ]

	Rose 2 Array(3) [ "Tallac", "Ralston", "Rose" ]
